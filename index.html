О проекте

Привет, сегодня я научу вас работать в HTML.

Чтобы начать работать в HTML, его сначала нужно включить.


                

                
Хорошо, HTML мы включили, теперь надо в нём начать работать.

                
В <head> можно написать название сайта с помощью тега <title>.

                
В <body> находится основное содержимое нашего сайта. Также чтобы написанный текст смотрелся красивее, нужно использовать теги.

                
Тег <p> создаёт параграф, тег <h1> создаёт заголовки, если же написать <h2>, <h3>, то будут создаваться подзаголовки.

                

                    

                

                
Также в HTML можно отцентрировать текст с помощью тега <center>. Чтобы текст отцентрировался, нужно заменить тег <p> на <center>.

                

                    

                

                
Ещё в <body> можно сделать рабочие ссылки. Для этого используется тег <a>.

                

                    

                

                
В кавычки нужно вставить саму ссылку, а перед концом ссылки (</a>) нужно написать название ссылки, то есть как оно будет отображаться на странице сайта.

                
Заключать ссылку в тег <p> нужно, чтобы ссылка была видна.

                
В HTML можно сделать текст подчёркнутым, используя тег <u>.

                

                    

                

                
Чтобы в HTML было легче работать, я оставил ссылки на полезные сайты.
